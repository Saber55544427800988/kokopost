# Koko Teacher Video Format Specification

## Overview
All videos generated by Koko Teacher follow a standardized 3-slide vertical format optimized for mobile viewing and social media platforms (YouTube Shorts, Instagram Reels, TikTok).

---

## Video Specifications

### Technical Requirements
- **Resolution**: 1080x1920 (vertical/portrait)
- **Duration**: 30-45 seconds
- **Frame Rate**: 24 fps
- **Codec**: H.264 (libx264)
- **Audio Codec**: AAC
- **Background**: Dark gradients (professional, eye-friendly)
- **Font**: Large, bold, sans-serif (Liberation Sans Bold / Arial Bold)

### Design Principles
- **Mobile-First**: Optimized for smartphone screens
- **Readability**: Large text, high contrast, minimal clutter
- **Professional**: Clean, consistent branding
- **Accessible**: Clear audio narration for all content

## Audio/TTS Configuration

### Voice Selection (All Male Voices)
- **English**: `en-US-GuyNeural` (Male, American)
- **Spanish**: `es-ES-AlvaroNeural` (Male, European Spanish)
- **French**: `fr-FR-HenriNeural` (Male, French)
- **Chinese**: `zh-CN-YunxiNeural` (Male, Mandarin)

### Audio Script Structure
The audio script alternates between:
1. **English narration** (explains, introduces, guides)
2. **Native language pronunciation** (demonstrates target language)

### Timing Guidelines
- **Slide 1**: ~5-10 seconds (intro + topic announcement)
- **Slide 2**: ~15-20 seconds (3 words Ã— ~5-7 seconds each)
- **Slide 3**: ~15-20 seconds (3 examples Ã— ~5-7 seconds each)
- **Total**: 30-45 seconds

### âœ… DO:

- Show target language words **with English meanings**
- Keep slides **short** (2-3 lines max per item)
- Use **large, bold fonts** for readability
- Maintain **consistent branding** (@KokoTeacher)
- Use **male TTS voices** for all narration
- Keep total duration **30-45 seconds**
- Use **dark/gradient backgrounds** for eye comfort
- Ensure **high contrast** (white/yellow text on dark backgrounds)

### âŒ DON'T:
- Don't use emojis in JSON output
- Don't clutter slides with too much text
- Don't use small fonts (minimum 32px)
- Don't exceed 3 items per slide
- Don't use bright/flashy backgrounds
- Don't mix female and male voices
- Don't create videos longer than 45 seconds
- Don't use light backgrounds

## Implementation Files

### Core Video Generation
- **`video_generator.py`**: Main lesson video generator (automated daily lessons)
- **`generate_special_video.py`**: Special/custom lesson video generator

### Supporting Files
- **`lesson_generator.py`**: AI-powered lesson content generation (Gemini/Groq)
- **`config.py`**: Video settings, API keys, scheduling
- **`main.py`**: Automated lesson cycle orchestration

### Both files implement:
1. `create_slide1()` - Topic introduction slide
2. `create_slide2()` - Vocabulary slide (3 words)
3. `create_slide3()` - Examples slide (3 phrases)
4. `create_gradient_background()` - Gradient background generator
5. `generate_tts_segment()` - TTS audio generation
6. `create_video()` / `main()` - Complete video assembly

## Quality Checklist

Before publishing any video, verify:

- [ ] Resolution is exactly 1080x1920 (vertical)
- [ ] Duration is between 30-45 seconds
- [ ] All text is large and readable on mobile
- [ ] Background gradients are dark and professional
- [ ] All 3 slides follow the exact layout format
- [ ] Branding (@KokoTeacher) appears on all slides
- [ ] English is the primary language on slides
- [ ] Target language words include English meanings
- [ ] Audio uses male voices only (English + native)
- [ ] Audio narrates all English text and target language words
- [ ] No emojis in the actual video content
- [ ] Video file size is reasonable (< 10MB)
- [ ] Audio is clear and synchronized with slides

## Maintenance Notes

### When updating the format:
1. Update **both** `video_generator.py` and `generate_special_video.py`
2. Update this documentation file
3. Test with all 3 languages (Spanish, French, Chinese)
4. Verify mobile viewing experience
5. Check audio synchronization
6. Validate video duration (30-45s)

### Common Issues:
- **Font not found**: Check font paths in `get_font_path()`
- **Audio desync**: Adjust slide duration calculations in `create_video()`
- **Text overflow**: Reduce font size or text length
- **Video too long**: Reduce audio script or slide durations
